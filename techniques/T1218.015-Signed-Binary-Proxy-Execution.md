# Objective

The objective of this Atomic Red Team test was to simulate the abuse of the legitimate Windows utility **rundll32.exe** to execute code while blending in with normal operating system activity.

Attackers frequently abuse trusted Microsoft-signed binaries, commonly referred to as **LOLBins (Living Off the Land Binaries)**, to evade detection and execute malicious actions without introducing obvious malware.

# Investigation Notes

Rundll32 is a legitimate Microsoft Windows utility designed to execute exported functions from Dynamic Link Libraries (DLLs).

Because the binary is digitally signed and commonly present on all Windows systems, attackers frequently abuse it to:

- Execute malicious DLLs
- Bypass application allow-listing
- Blend malicious activity with legitimate Windows processes
- Evade security monitoring

Although the execution performed during this lab was part of a controlled Atomic Red Team simulation, it generated endpoint telemetry similar to activity observed during real-world intrusions.

- Legitimate Windows binaries can be abused to execute malicious code.
