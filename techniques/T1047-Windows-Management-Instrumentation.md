# Objective

The objective of this Atomic Red Team test was to simulate process execution through Windows Management Instrumentation (WMI) and observe the resulting endpoint telemetry using Sysmon and LimaCharlie.

WMI is a legitimate Windows management framework frequently abused by attackers to execute commands, perform remote administration, and move laterally across enterprise environments.

# Execution Outcome

The Atomic test did **not execute successfully** because the `wmic` utility is no longer included by default in modern Windows 11 installations.

Although the intended simulation could not complete, the attempted execution still demonstrates an important defensive consideration: older attack techniques may fail on modern operating systems due to deprecated or removed binaries.

# Investigation Notes

Windows Management Instrumentation has historically been one of the most commonly abused Windows administration technologies.

Attackers frequently leverage WMI to:

- Execute local commands
- Execute remote commands
- Perform lateral movement
- Launch PowerShell
- Execute malware without directly spawning user applications

In this lab, the Atomic test could not complete because the **WMIC** command-line utility has been deprecated and removed from newer Windows 11 versions.

This behaviour itself is useful to document, as it demonstrates how attack simulations can be affected by differences between Windows versions.
