# Objective

The objective of this Atomic Red Team test was to simulate PowerShell-based command execution and observe how the activity is recorded by Sysmon and LimaCharlie.

PowerShell is one of the most frequently abused Windows utilities during real-world attacks due to its powerful scripting capabilities and native availability on Windows systems.

# Investigation Notes

PowerShell is widely used by both administrators and threat actors.

Because PowerShell provides direct access to the Windows operating system and .NET framework, it is frequently abused for:

- Script execution
- Downloading payloads
- Discovery
- Credential access
- Defense evasion

Although this Atomic test executed a harmless PowerShell command, the generated telemetry closely resembles the type of activity analysts investigate during real incidents.
